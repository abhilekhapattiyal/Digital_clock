<h1>Digital Clock</h1></br>
<h3>Description</h3></br>
This project is a simple Digital Clock built using Python and the Tkinter library. The clock displays the current time in hours:minutes
format and updates in real-time. It's a beginner-friendly project that demonstrates the usage of GUI programming with Tkinter and time management with the time module in Python.

<h3>Features</h3></br>
<ul>
<li>Displays the current time (HH:MM
) in a digital clock format.</li>
<li>Updates the time every second.</li>
<li>Customizable clock style with options for font and background colors.</li>
<li>User-friendly and easy to modify.</li>
</ul>
<h3>Technologies Used</h3></br>
<ul>
<li>Python: Programming language used to develop the clock.</li>
<li>Tkinter: Python's standard library for creating graphical user interfaces (GUI).</li>
<li>Time module: Used to get the current system time and update it.</li>
</ul>
